# Deploy Project Lab

Website belajar deploy project web ke hosting untuk pemula yang ingin melanjutkan pembelajaran HTML, CSS, JavaScript, PHP, Laravel, dan GitHub ke workflow profesional.

## Fitur

- 20 materi bertahap tentang domain, cPanel, database, upload ZIP, phpMyAdmin, koneksi PHP, Laravel, testing, debugging, backup, keamanan, dan handover
- Halaman detail terpisah untuk setiap materi di folder `materi/`
- Quiz interaktif
- Latihan debugging hosting
- Recall konsep
- Simulator checklist deploy cPanel, koneksi database, dan catatan phpMyAdmin
- Mini project deploy
- Progress tracker dengan `localStorage`
- Dark mode
- Hero image bertema deploy hosting
- Tema warna dasar biru muda

## Cara menjalankan

Buka `index.html` langsung di browser, atau publish folder ini ke GitHub Pages. Daftar materi ada di `materi.html`, sedangkan isi setiap materi ada di folder `materi/`.

## Catatan keamanan

Contoh cPanel, database, user, dan password di materi memakai placeholder. Jangan menaruh password asli, URL login cPanel pribadi, file SQL backup, atau file ZIP source project di repository publik.
